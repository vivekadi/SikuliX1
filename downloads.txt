<hr><h4>Needed for scripting support in the IDE and running scripts from commandline</h4>

<h4><a href="https://repo1.maven.org/maven2/org/python/jython-standalone/2.7.1/jython-standalone-2.7.1.jar">The Jython interpreter 2.7.1 for python scripting (the default)</a></h4>
<h4><a href="https://repo1.maven.org/maven2/org/jruby/jruby-complete/9.2.0.0/jruby-complete-9.2.0.0.jar">The JRuby interpreter 9.x for ruby scripting</a></h4>
Download what you prefer or both to the same folder as you have your sikulix.jar.
<br>They will be recognized by SikuliX at the next start.
<br>So preferably download before using the IDE the first time.

<hr><h4><a href="https://raiman.github.io/SikuliX1/sikulix.jar">Download the ready to use sikulix.jar (SikuliX IDE)</a></h4>
Just download, move it to a folder of your choice and doubleclick to run it.<br>
An alternative to run it is from commandline using
<pre>java -jar path-to/sikulix.jar</pre>
